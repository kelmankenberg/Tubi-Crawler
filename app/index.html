Help & Getting Started

        
      

      

        

          
📱 What is Tubi Crawler?

          
Tubi Crawler is a desktop application that extracts episode URLs from Tubi TV series pages. It helps you quickly gather all episode links from any series into a single text file.

        

        
        

          
🚀 Quick Start

          
    
            
  1. Copy a Tubi series URL (e.g., https://tubitv.com/series/3155)
    2. 
            
  2. Paste it into the URL input field
    3. 
            
  3. Click "Crawl" or press Enter
    4. 
            
  4. URLs will appear in the text area below
    5. 
            
  5. Use "Save As" to download as a text file
    6. 
          

        


        

          
💡 Key Features

          
    
            
  • URL Validation: Real-time check if your URL is a valid Tubi series link
    • 
            
  • Quick Actions: Clipboard button to paste & crawl, Clear & Crawl for fresh results
    • 
            
  • Recent URLs: Access previously crawled series from the Recent button
    • 
            
  • Status Indicators: Color-coded logs show success/error/warning status
    • 
            
  • Episode Counter: See how many URLs were found and crawl time
    • 
            
  • YT-DLP Integration: Download episodes directly (requires configuration)
    • 
          

        


        

          
⚙️ Settings

          
Open Settings (Ctrl+,) to configure:

          
    
            
  • Theme: Light, Dark, or System-based
    • 
            
  • Preferred Browser: Use external or internal browser
    • 
            
  • YT-DLP Settings: Download location, format, merge options
    • 
          

        


        

          
❓ FAQ

          
Q: The URL isn't validating. What's wrong?

          
A: Make sure the URL follows this format: https://tubitv.com/series/xxxx or similar

          
          
Q: How do I download episodes?

          
A: Use the YT-DLP button after crawling. First configure it in Settings

          
          
Q: Where are my files saved?

          
A: "Save As" uses the standard file dialog. YT-DLP downloads go to the configured folder
