# Features

- Input field for Tubi series URL
- Automated crawl to retrieve all episodes
- Display of detected seasons and episodes
- Export to `.txt` file with full URLs
- Optional advanced mode using JSON API if accessible
- Cross-platform (Windows/macOS/Linux) via Electron

## Planned Enhancements
- CSV and JSON export options
- Episode thumbnail retrieval
- Error logging and retry mechanism
- Built-in URL validation and status feedback
