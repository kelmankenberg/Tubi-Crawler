# UI Layout Plan

## Main Window
| Element | Description |
|----------|--------------|
| URL Input | Text field for the main series URL |
| Crawl Button | Starts the crawling process |
| Log Output | Displays crawling progress and results |
| Export Button | Saves results to a text file |
| Status Bar | Shows success/failure messages |

## Style Guide
- Minimalist Electron window with dark/light theme toggle
- Font: Inter / system default
- Rounded inputs and buttons
- Optional advanced view for debugging
