# User Stories

### End User
- **As a user**, I want to enter a Tubi show URL so I can get a list of all episodes.
- **As a user**, I want the app to automatically detect all seasons and episodes.
- **As a user**, I want to export episode data to a text file for offline viewing.
- **As a user**, I want clear progress updates during the crawl.

### Developer
- **As a developer**, I want modular crawling logic to easily adapt to other sites.
- **As a developer**, I want logging to detect failed URLs.
- **As a developer**, I want a local configuration file for debugging API endpoints.
