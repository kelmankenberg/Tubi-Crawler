- Set up Electron shell and renderer
- Add basic UI (URL input + crawl button)
- Integrate Puppeteer for initial data extraction

## Phase 2 – Core Logic
- Identify and parse JSON episode data
- Handle pagination and dynamic requests
- Output structured text file

## Phase 3 – UX Enhancements
- Add progress display
- Validate URLs before crawl
- Implement export options

## Phase 4 – Packaging & Testing
- Build for Windows, macOS, and Linux
- Add logging and error handling
- Conduct functional testing across 3–5 series URLs
