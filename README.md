# Tubi Series Episode Crawler

An Electron + Node.js desktop app that crawls Tubi TV series pages to extract episode data and export it as a text file.

## Overview
The app takes a Tubi series URL (e.g., `https://tubitv.com/series/300004778/heartbeat`) and retrieves all episode titles, season numbers, and URLs. Output is saved as a formatted `.txt` file for offline reference.
